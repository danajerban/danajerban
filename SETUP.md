# My Development Setup

## Daily Drivers

- [**Ghostty**](https://ghostty.org/) — Fast, native terminal emulator built in Zig
- [**Zed**](https://zed.dev/) — High-performance code editor written in Rust
- [**tmux**](https://github.com/tmux/tmux) — Terminal multiplexer for persistent sessions and split panes
- [**Oh My Zsh**](https://ohmyz.sh/) — Zsh framework with Dracula theme
- [**Yazi**](https://github.com/sxyazi/yazi) — Blazing fast terminal file manager written in Rust
- [**Zoxide**](https://github.com/ajeetdsouza/zoxide) — Smarter `cd` that learns your most-used directories
- [**Espanso**](https://espanso.org/) — Open-source OS-level snippets written in Rust
- [**OrbStack**](https://orbstack.dev/) — Docker and Kubernetes on macOS

## AI Development

- [**Claude Code**](https://docs.anthropic.com/en/docs/claude-code/overview) — Anthropic's agentic CLI for Claude
- [**Factory Droid CLI**](https://factory.ai) — AI coding agent with multi-model support — same setup as Claude Code
- [**Context7**](https://github.com/upstash/context7) MCP — Up-to-date library and framework documentation for any LLM
- [**RTK**](https://github.com/rtk-ai/rtk) — Rust CLI proxy that reduces LLM token consumption by ~80%
- [**VibeProxy**](https://github.com/automazeio/vibeproxy) — macOS app to route coding agents through existing AI subscriptions, built on [CLIProxyAPIPlus](https://github.com/router-for-me/CLIProxyAPIPlus)
